# About

Fullstack project made in nextjs with CRUD operations

# Technologies and Tools

- NextJs 13.4
- Next Auth
- Tailwind CSS
- TypeScript
- Mongoose
- MongoDB

# ENV NEEDED

- MONGODB_URI: Database URI
- NEXTAUTH_SECRET: any secret string
- NEXTAUTH_URL: deployment or localhost url
- NEXTAUTH_URL_INTERNAL: localhost url

- GOOGLE_CLIENT_ID: google client id generated from google developer console
- GOOGLE_CLIENT_SECRET: google client secret generated from google developer console
